# Terminologi Penting

## Web Platform

Web Platform adalah kumpulan spesifikasi yang mendefinisikan bagaimana browser menyediakan API bagi aplikasi web.

## Web APIs

Web APIs adalah antarmuka yang disediakan oleh browser untuk memungkinkan JavaScript berinteraksi dengan lingkungan browser.

Contohnya:

- DOM API
- Fetch API
- Storage API
