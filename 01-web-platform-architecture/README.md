# Web Platform Architecture

Buku ini membahas arsitektur dasar Web Platform dan hubungan antar komponen yang membentuk lingkungan eksekusi web di browser.

Topik yang dibahas dalam buku ini meliputi:

- ekosistem spesifikasi Web Platform
- hubungan antara ECMAScript dan Web APIs
- peran HTML Standard dalam mendefinisikan runtime browser
- hubungan antara JavaScript engine dan browser engine

---

# Struktur Buku

Buku ini terdiri dari beberapa bagian utama:

1. Introduction  
2. Core Concepts  
3. Runtime Model  
4. Examples

---

# Tujuan Buku

Setelah membaca buku ini, pembaca diharapkan memahami:

- apa itu Web Platform
- bagaimana spesifikasi web saling berhubungan
- bagaimana browser menyediakan Web APIs
- bagaimana JavaScript berinteraksi dengan Web Platform
