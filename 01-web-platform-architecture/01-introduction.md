# Latar Belakang

Web Platform adalah lingkungan eksekusi yang memungkinkan aplikasi web berjalan di browser.

Banyak developer menganggap bahwa fitur seperti DOM, Fetch, atau timers adalah bagian dari JavaScript.

Namun sebenarnya fitur tersebut berasal dari **Web Platform APIs**, bukan dari bahasa JavaScript itu sendiri.
