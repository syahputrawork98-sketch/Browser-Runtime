# Contributing Guide

Terima kasih telah tertarik untuk berkontribusi pada repository **Browser Runtime**.

Repository ini bertujuan menjadi **knowledge base jangka panjang tentang browser runtime dan Web Platform**.

---

# Prinsip Kontribusi

Kontribusi harus:

* akurat secara teknis
* selaras dengan spesifikasi Web Platform
* tidak menimbulkan miskonsepsi

---

# Struktur Penulisan

Setiap topik baru harus memiliki struktur berikut:

1. Official Term
2. Definisi
3. Mental Model
4. Cara Kerja Runtime
5. Contoh Kode
6. Common Misconceptions
7. Pitfalls

---

# Bahasa

Penulisan menggunakan:

* Bahasa Indonesia untuk penjelasan
* Bahasa Inggris untuk istilah teknis

Contoh:

```
Event Loop
Task Queue
Microtask Queue
Rendering Pipeline
```

---

# Menambahkan Topik Baru

Topik baru harus ditempatkan di domain yang sesuai.

Contoh:

Event loop → `03-event-loop-and-scheduling`

---

# Perubahan Struktur

Perubahan struktur repository harus didiskusikan terlebih dahulu.

---

# Tujuan Repository

Repository ini dirancang untuk menjadi:

* referensi teknis
* roadmap belajar
* dokumentasi Web Platform
