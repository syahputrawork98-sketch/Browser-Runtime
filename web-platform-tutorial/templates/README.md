# Templates - Web Platform Tutorial

Template di folder ini adalah sumber utama untuk penulisan materi track `web-platform-tutorial`.

## Cara Pakai
1. Mulai dari `core-topic-template.md`.
2. Tambahkan section khusus dari `web-platform-tutorial-template.md` (A-D wajib).
3. Simpan topik baru ke folder level yang sesuai (`../beginner/`, `../intermediate/`, `../advanced/`, atau `../expert-bridge/`) dengan prefix angka.

## Aturan Level
- Semua aturan level dipusatkan di `../README.md` section `Aturan Level (Ringkas)`.
