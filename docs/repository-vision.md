# Repository Vision

Repository **Browser Runtime** adalah koleksi buku teknis yang bertujuan menjelaskan bagaimana browser bekerja secara internal ketika menjalankan JavaScript dan Web Platform APIs.

Fokus utama repository ini adalah memahami Web Platform dari perspektif:

- runtime
- arsitektur sistem
- spesifikasi web

Repository ini tidak berfokus pada framework frontend, tetapi pada sistem yang membuat framework tersebut dapat berjalan.

Tujuan jangka panjang repository ini adalah menjadi referensi teknis untuk memahami:

- bagaimana JavaScript dijalankan di browser
- bagaimana Web APIs bekerja
- bagaimana event dijadwalkan
- bagaimana rendering terjadi
- bagaimana networking diproses
- bagaimana security boundary diterapkan
