# Repository Architecture

Repository ini memiliki dua jenis konten utama:

1. Buku
2. Dokumentasi repository

Struktur utama repository:

browser-runtime/
│
├── docs/
│
├── 01-web-platform-architecture/
├── 02-javascript-engine-and-execution-model/
├── 03-browser-runtime-fundamentals/
├── 04-dom-and-event-system/
├── 05-browser-rendering-pipeline/
├── 06-browser-networking-and-fetch/
├── 07-browser-storage-systems/
├── 08-browser-concurrency-and-workers/
├── 09-browser-security-model/
├── 10-browser-process-architecture/
├── 11-web-standards-companion/
├── 12-browser-runtime-case-studies/
└── 13-runtime-experiments-and-katas/

Folder `docs/` berisi dokumentasi tentang repository itu sendiri.

Folder bernomor berisi buku.
