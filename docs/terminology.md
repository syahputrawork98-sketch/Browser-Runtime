# Terminology

Beberapa istilah penting dalam repository ini mengikuti terminologi resmi Web Platform.

## Event Loop

Mekanisme yang digunakan browser untuk menjadwalkan task.

## Task

Unit pekerjaan dalam event loop.

## Microtask

Queue khusus untuk pekerjaan yang harus dijalankan sebelum task berikutnya.

## EventTarget

Interface dasar untuk objek yang dapat menerima event.

## Rendering Pipeline

Proses yang digunakan browser untuk merender halaman:

style → layout → paint → composite
