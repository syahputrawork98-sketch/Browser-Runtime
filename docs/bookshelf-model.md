# Bookshelf Model

Repository ini menggunakan model **bookshelf**.

Repository bukan satu buku, tetapi **rak buku** yang berisi beberapa buku tentang Browser Runtime.

Struktur konseptualnya:

Repository  
→ rak buku

Folder bernomor  
→ buku

Markdown di dalam folder  
→ bab buku

Contoh:

browser-runtime/
│
├── 01-web-platform-architecture/
├── 02-javascript-engine-and-execution-model/
├── 03-browser-runtime-fundamentals/

Setiap folder merepresentasikan **satu buku dengan topik tertentu**.

Model ini memungkinkan repository berkembang tanpa kehilangan struktur yang jelas.
