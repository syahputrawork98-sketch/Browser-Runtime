# Writing Guidelines

Dokumentasi dalam repository ini mengikuti beberapa prinsip utama.

## Terminologi mengikuti spesifikasi

Istilah yang digunakan harus mengikuti spesifikasi resmi seperti:

- HTML Standard
- DOM Standard
- Fetch Standard
- ECMAScript Specification

## Fokus pada Runtime

Penjelasan harus menjawab pertanyaan seperti:

- berjalan di thread mana
- bagaimana dijadwalkan
- queue apa yang digunakan
- kapan rendering terjadi

## Hindari penjelasan berbasis framework

Repository ini menjelaskan **Web Platform**, bukan framework seperti React atau Vue.

Framework hanya boleh disebut sebagai contoh jika diperlukan.
