# Browser Runtime

Repository ini adalah **rak buku teknis** yang berisi kumpulan buku tentang **Browser Runtime dan Web Platform**.

Setiap folder dalam repository ini merepresentasikan **sebuah buku** yang membahas aspek tertentu dari bagaimana browser bekerja secara internal.

Dokumentasi dalam repository ini berfokus pada pemahaman **Web Platform dari perspektif runtime dan arsitektur browser**.

---

# Tujuan Repository

Repository ini bertujuan membantu developer memahami bagaimana browser bekerja secara internal, termasuk:

* bagaimana JavaScript dijalankan
* bagaimana event dijadwalkan
* bagaimana rendering terjadi
* bagaimana networking diproses
* bagaimana security boundary diterapkan

Pendekatan yang digunakan adalah **arsitektur sistem dan spesifikasi Web Platform**.
