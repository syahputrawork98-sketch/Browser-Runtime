# Templates

Folder ini berisi template standar penulisan materi Browser Runtime.

## Cara Pakai
1. Mulai dari `core-topic-template.md`.
2. Pilih satu template track sesuai folder target.
3. Gabungkan isi `core` dan `track-specific` saat menulis topik.

## Daftar Template
- `core-topic-template.md`
- `web-platform-tutorial-template.md` (dipindah ke `web-platform-tutorial/templates/`)
- `browser-runtime-first-principles-template.md`
- `browser-engine-and-rendering-book-template.md`
- `runtime-exercises-and-katas-template.md`
- `browser-case-studies-template.md`
- `web-standards-spec-companion-template.md`
